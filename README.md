Project
------------

On the back-end,
- app starts at `server.js`
- add frameworks and packages in `package.json`
- safely store app secrets in `.env`

On the front-end,
- edit `client.js`, `style.css` and `index.html`
- drag in `assets`, like images or music, to add them to your project
